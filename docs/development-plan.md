# 开发任务计划表（15 周 / 单人全栈）

> 开发人员：1 人，兼任前端、后端、测试。
> 总工期：15 周（一期全部功能）。
> 二期功能不在本计划内，在一期完成后单独排期。
> 每周按 5 个工作日计算。
> 说明：本文档为历史开发计划归档，企微入口形态、权限层级与 ASR 实现请以 `docs/project-status.md` 和 `docs/deployment.md` 为准。当前正式移动入口为**企微工作台 H5**，企微侧边栏方案已归档，不再作为主链路。

## 范围说明

### 一期范围（本计划覆盖）

- 后端 API 全部一期功能
- PC 后台前端全部一期功能
- 企微工作台 H5（当前正式入口）
- 企微侧边栏 H5（历史方案，已归档）
- 基础设施与部署

### 明确不含（需外部条件或属于二期）

| 内容 | 原因 |
|------|------|
| 微信小程序（录音遥控器） | 依赖硬件方协议和蓝牙对接，需硬件就绪后单独开发 |
| AI 上下文关联质检 | 二期 |
| 多维深化质检评分 | 二期 |
| 客户深层标签 / 多维画像 | 二期 |
| 客户盘点看板 | 二期 |
| 客户追踪详情页（多次到访深度分析） | 二期 |
